High Order Function

Itérer: forEach (pourChaque)

La méthode arr.forEach permet d’exécuter une fonction pour chaque élément du tableau.
Elle conserve le tableau d'entrée! 


let prenoms=['Alma','Coline','Edmond']
prenoms.forEach((elem,idx)=> {
  prenoms[idx]=elem.toUpperCase()
})
console.log(prenoms)

Résultat

Itérer: map

La méthode arr.map est l’une des plus utiles et des plus utilisées. Elle appelle la fonction pour chaque élément du tableau et retourne le tableau de résultats.


let prenoms=['Alma','Coline','Edmond']
let majPrenoms = prenoms.map((elem)=> {
  return elem.toUpperCase()
})
console.log(majPrenoms)

Résultat

Trier: sort

La méthode arr.sort permet de trier un tableau en fonction d'une méthode de trie
le tableau d'entrée est modifié!


let prenoms=['Edmond','Alma','Medhi']
prenoms.sort((a,b)=> {
  return a.localeCompare(b)
})

console.log(prenoms)

Filtrer: filter

La méthode arr.filter permet d'exclure via un filtre des éléments d'un tableau
filter renvoie un nouveau tableau!


let notes = [18, 10, 5, 2, 14];
let onlyMoyenne = notes.filter((note) => note >= 10);

console.log(onlyMoyenne);
